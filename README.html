{ggstatsplot}: R Package for {ggplot2} Based Plots with Statistical Details

This is a repository for presentation about the ggstatsplot package (prepared using R package xaringan).

The slides are available here:
https://indrajeetpatil.github.io/ggstatsplot_slides/slides/ggstatsplot_presentation.html#1

There is also a dedicated website for this package:
https://indrajeetpatil.github.io/ggstatsplot/

Follow Along

You can use the script ggstatsplot_presentation.R to follow along during the talk or to review the plots later.

Setup

You will need to install the following package-

install.packages("ggstatsplot")
